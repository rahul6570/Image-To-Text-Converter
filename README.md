# Image-To-Text-Converter

### An app made to generate text corresponding to image scanned via camera or taken from gallery.

## TechStacks & Library used:
#### JAVA
#### Android Studio 
#### Firebase ML Kit
#### ImagePicker Library ( https://github.com/Dhaval2404/ImagePicker )
